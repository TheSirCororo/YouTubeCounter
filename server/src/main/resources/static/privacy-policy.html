Privacy Policy

This application uses Google Account authentication and requests the following OAuth scope:

  • https://www.googleapis.com/auth/youtube.readonly

This scope is used to retrieve public YouTube video statistics (such as views and likes) for use in displaying translation engagement data.

Access and refresh tokens are stored temporarily on the server in order to maintain user sessions and securely refresh tokens. Tokens are not stored persistently, are not shared with third parties, and are protected using standard security measures.

We do not collect or store any personal information beyond what is necessary for authentication and accessing public YouTube data. All user data is handled securely and used solely for the intended functionality of the application.
